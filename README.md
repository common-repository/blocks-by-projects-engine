# GoodBlocks by Projects Engine

GoodBlocks is a powerful WordPress plugin designed to enhance your Gutenberg editor experience. It offers a wide variety of customizable and versatile blocks, allowing you to create engaging content with ease.

## Description

GoodBlocks by Projects Engine is packed with features to supercharge your WordPress website. Whether you're looking to create eye-catching call-to-action sections, design intricate layouts, or add unique interactive elements, GoodBlocks has you covered. Each block is crafted with flexibility in mind, making it easy to adjust styles, colors, and settings to suit your needs.

## Features

- **Customizable Blocks:** A variety of Gutenberg blocks with extensive customization options.
- **Responsive Design:** All blocks are fully responsive and adapt seamlessly to different screen sizes.
- **Easy to Use:** Intuitive controls and settings that integrate smoothly with the Gutenberg editor.
- **Regular Updates:** Continuous updates to ensure compatibility with the latest WordPress versions.

## Installation

1. Download the plugin from the [GoodBlocks by Projects Engine website](https://projectsengine.com/plugin/good-blocks).
2. In your WordPress dashboard, go to **Plugins** > **Add New** > **Upload Plugin**.
3. Upload the `goodblocks.zip` file and click **Install Now**.
4. Activate the plugin through the **Plugins** menu in WordPress.
5. Start creating with GoodBlocks in the Gutenberg editor.

## Requirements

- WordPress version 6.5 or higher.
- PHP version 7.4 or higher.

## Usage

1. After activation, go to any page or post in the Gutenberg editor.
2. Click the **Add Block** button (`+`) and look for the **GoodBlocks** category.
3. Select the desired block and customize it using the block settings panel.
